
# Georgian Playback Theatre Center — Website Concept & Technical Specification

Author: Prepared for project planning
Language: Georgian / English mix where needed

---

# 1. Project Vision

The website should represent **Playback Theatre not just as information, but as an experience.**

Goal:

Visitors → Emotional connection → Interest → Registration

The design should feel:

- minimal
- calm
- human
- theatrical
- emotionally authentic

---

# 2. Design Philosophy

Playback Theatre is about:

- listening
- human stories
- presence
- authenticity
- emotional reflection

The website must visually communicate:

• silence  
• space  
• listening  
• emotional depth  

Design style:

Minimalist editorial theatre style.

Large typography  
Lots of whitespace  
Few colors  
Emotional photography

---
