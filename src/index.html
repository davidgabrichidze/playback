სექციების მიმდევრობა - AIDA

სექციები დალაგებულია AIDA პრინციპით (Attention → Interest → Desire → Action) - ჯერ ყურადღება, მერე ინტერესი, მერე სურვილი, მერე მოქმედება.

Attention Hero - პირველი ემოცია
Interest What + How - რა არის და როგორ მუშაობს
Desire Impressions → Founder → Team → Gallery - ნდობა, ემოცია, ვიზუალი
Action Performance → Donation → Organizations → Courses - მოქმედება

ცისიას ხმა საიტზე

ცისია ჩოლოყაშვილი, თანადამფუძნებელი, საიტზე რამდენჯერმე ჩნდება - და ეს განზრახ არის ასე. ის არ არის „ციტატა გვერდის გასალამაზებლად". ცისია სტუმარს უყვება თავის ამბავს: ჯერ ხსნის რატომ გადაწყვიტა ამ თეატრის შექმნა (Impressions), მერე ყვება ცენტრის ისტორიას (Founder), და ბოლოს აღწერს რა ხდება ყოველ წარმოდგენაზე (Team). სტუმარი შემოსულია არა უბრალოდ ვებსაიტზე, არამედ იცნობს და უსმენს ადამიანს, რომელიც ამ ყველაფრის უკან დგას.
