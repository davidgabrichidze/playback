## 4. ტექსტის ტონი — შენობა → თქვენობა

უშუალობა უნდა შენარჩუნდეს, დისტანცია არ უნდა გაიზარდოს, მაგრამ ფამილიარულობა მოიხსნას.

### 4a. „როგორ მუშაობს” სექცია — 3 ნაბიჯის გადაწერა

ახლანდელი (ორივე HTML-ში ერთნაირია):
```
1. შენ გვიყვები → რაც გაზიარებადია — პირადი, რეალური, შენი ამბავი
2. ჩვენ ვასრულებთ → შენი ისტორია იქვე თამაშდება — მოძრაობით, მუსიკით, ემოციით, სიტყვებით
3. ყველა განიცდის → ერთი ისტორია — საერთო გამოცდილება
```

ახალი (ცისიას ხაზით, მაგრამ ისევ „თვალებში ვუყურებთ მთხრობელს”):
```
1. ვუსმენთ → მაყურებელი, რომელიც მოისურვებს, გვიყვება რეალურ ამბავს თავისი ცხოვრებიდან
2. ვთამაშობთ → ამბავი იქვე თამაშდება მსახიობების მიერ, ცოცხალი მუსიკის თანხლებით, მოძრაობებით, სპონტანურად
3. ვუბრუნებთ (playback) → ამ ამბავს რიტუალურად ვუბრუნებთ მთხრობელს და ის საერთო გამოცდილება ხდება
```

EN ვერსიაც შესაბამისად:
```
1. We listen → A member of the audience shares a real story from their life
2. We play → The story is performed right there by actors, with live music
3. We return (playback) → We ritually return this story to the teller, and it becomes a shared experience
```

## 5. შთაბეჭდილებები — ახალი ციტატების დამატება

ცისიას უკუკავშირიდან ერთი შეცვლილი და ერთიც ახალი ციტატა:

ახლანდელი მაყურებლის ციტატები (ორივე HTML-ში):
1. „პირველად ვიყავი და ვერ წარმოვიდგენდი, რომ ასეთი რამ შეიძლებოდა. ტირილიც მინდოდა და სიცილიც.”

3. „ამ საღამოს შემდეგ მთელი კვირა სხვა ადამიანი ვიყავი.”


პირველი ახალი ციტატა არსებულ მესამეს ანაცვლებს:
- „ამ საღამოს შემდეგ მთელი კვირა სხვა ადამიანი ვიყავი” → „ამ საღამოს შემდეგ მთელი კვირა ემოციებით სავსე ვიყავი”
