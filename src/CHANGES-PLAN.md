**დასამატებელი:** მოკლე აბზაცი ცისიას ციტატამდე ან მის შემდეგ, რომელიც აღწერს:

## 2. კურსები/ორგანიზაციები/ეთიკა — თეატრალურის სტრუქტურა + width

თეატრალურში (alternative.html) ამ სექციებს უკეთესი HTML სტრუქტურა აქვს:
- ორგანიზაციებში `services-grid` + `service-item` (h3 + p), სინემატიკში `org-services` + `org-service` (h4 + p)
- კურსებში თეატრალურს აქვს `h3` + `course-duration` + `p`, სინემატიკში `course-name` + `course-duration` + `course-desc`
- ეთიკაში თეატრალურს აქვს `h3` + `p`, სინემატიკში accordion `data-index` + `ethics-title` + `ethics-icon`

**გასაკეთებელი:** სინემატიკში (index.html) ამ სამი სექციის HTML მიმსგავსება თეატრალურის სტილს, **+ width გაზრდა**.

ახლანდელი `.section-content` default: `max-width: 800px`
კურსები: `max-width: 1000px` (ხაზი ~690)
ორგანიზაციები: `max-width: 900px` (ხაზი ~537) — **გასაზრდელი ~1000-1100px-მდე**
ეთიკა: `max-width: 900px` (ხაზი ~1032) — **გასაზრდელი ~1000px-მდე**

| ფაილი | ხაზი | ცვლილება |

## 3. ციტატების გადანაწილება

### 3b. „პირადად ჩემთვის..." ციტატა — გუნდში ცისიას აღწერაში

> „პირადად ჩემთვის, როგორც ქართული ფლეიბექ თეატრის დამფუძნებლისა და ხელმძღვანელისთვის, ეს სივრცე წარმოადგენს ჩემი ორი პროფესიული გამოცდილების თანხვედრას: პროფესიული თეატრის და ფსიქოთერაპევტის პრაქტიკას."

ეს ზუსტად ხსნის ვინ არის ცისია და რატომ სწორედ ის ხელმძღვანელობს ამ სივრცეს.

## 4. ცისიას ატრიბუცია — ორსართულიანი კომპონენტი ყველგან

ყველგან სადაც ცისია ჩოლოყაშვილის ციტატაა, ავატარის გვერდით უნდა ჩანდეს:
- **მწკრივი 1:** სახელი (ცისია ჩოლოყაშვილი)
- **მწკრივი 2:** რეგალიები მეორადი სტილით (დამფუძნებელი, ხელმძღვანელი)

### ახლანდელი HTML პატერნი (მაგ. impressions, ხაზი ~1873):
```html
<img class="avatar-small" ...>
<span>
  <span class="ka">ცისია ჩოლოყაშვილი</span>
  <span class="en">Tsisia Cholokashvili</span>
</span>
```

### ახალი HTML პატერნი:
```html
<img class="avatar-small" ...>
<div class="avatar-info">
  <span class="avatar-name">
    <span class="ka">ცისია ჩოლოყაშვილი</span>
    <span class="en">Tsisia Cholokashvili</span>
  </span>
  <span class="avatar-role">
    <span class="ka">დამფუძნებელი, ხელმძღვანელი</span>

  </span>
```
